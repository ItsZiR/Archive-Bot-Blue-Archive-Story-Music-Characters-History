Archive Bot: 紀載蔚藍檔案相關的OST, 劇情跟音樂等關聯，以及實裝歷史等資料的Discord Bot.
主要面向用戶為繁體中文圈，查詢內容會先以中文為主，但為嚴謹計算時間，所有日期均先以日服實裝時間為基準。
由Discord.py編寫, 資料庫使用SQLite.

<搭載功能>
音樂: 以編號查詢OST資訊、以作者查詢OST列表(隨機推播，待更新);
學生: 查詢目前實裝的學生數量、以學校統計特定學生佔的比例;
劇情: 查詢所有已實裝的劇情順序、主線劇情大全
(開發者專用)
Bot手動重開，不用關機 / 卸載或重新掛載模組

<使用說明>
使用以下URL邀請Bot:
https://discord.com/oauth2/authorize?client_id=1485930769825136710
可選擇讓bot用app的形式，就可以在任何群組使用，但有些群組會設定外部bot的訊息只有你自己能看到;
若選擇加到伺服器則可以使用如訊息回應等更多功能。
